# WPF Image Filter

A simple yet powerful WPF application for applying various filters to images in batch.

## Features

- **Batch Processing**: Load and process multiple images at once.
- **Drag & Drop**: Easily add images by dragging them into the application.
- **Filters**: Choose from a variety of effects:
  - Grayscale
  - Sepia
  - Warm
  - Cold
  - Blur
- **Real-time Progress**: Track processing status with a progress bar and log messages.
- **Non-destructive**: Saves processed images as new files with filter prefixes, keeping your originals safe.
- **Cancellation**: Stop processing at any time.
